py_stringmatching.tokenizers.delimiter(input_string, delim_str=' ')[source]¶

Tokenizes input string based on the given delimiter.

Parameters:
  • input_string (str) – Input string
  • delim_str (str) – Delimiter string
Returns:

Token list (list)
Raises:

TypeError – If the input is not a string

Examples

>>> delimiter('data science')
['data', 'science']
>>> delimiter('data$#$science', '$#$')
['data', 'science']
>>> delimiter('data science', ',')
['data science']

py_stringmatching.tokenizers.qgram(input_string, qval=2)[source]¶

Tokenizes input string into q-grams.

A q-gram is defined as all sequences of q characters. Q-grams are also known as n-grams and k-grams.

Parameters:
  • input_string (str) – Input string
  • qval (int) – Q-gram length (defaults to 2)
Returns:

Token list (list)
Raises:

TypeError – If the input is not a string

Examples

>>> qgram('database')
['da','at','ta','ab','ba','as','se']
>>> qgram('a')
[]
>>> qgram('database', 3)
['dat', 'ata', 'tab', 'aba', 'bas', 'ase']
